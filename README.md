# Promise Tracker

A full-stack application for tracking promises and commitments.

## Project Structure

- `mobile/` - React Native mobile app (Expo)
- `backend/` - Node.js/Express backend with PostgreSQL

## Getting Started

### Backend

1. Navigate to the backend directory:
```bash
cd backend
```

2. Install dependencies:
```bash
npm install
```

3. Set up your `.env` file (see backend README)

4. Start the server:
```bash
npm run dev
```

### Mobile

1. Navigate to the mobile directory:
```bash
cd mobile
```

2. Install dependencies:
```bash
npm install
```

3. Start the Expo development server:
```bash
npm start
```
